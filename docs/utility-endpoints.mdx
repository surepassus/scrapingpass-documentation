## Base URL

Utility endpoints are well separated from the actual API to prevent any conflicts to /meta/ path.

```
https://api.scrapingpass.com/meta
```

## Estimate Balance Required

**GET** `/estimate-balance`

Calculate and determine the exact balance which will be used for provided set of Scrapingpass API options. The parameters for this endpoint are exactly same as that of [HTTP REST API](/getting-started/scrapingpass-api#specifying-api-options).

:::note
This endpoint doesn't requires an API Key or any kind of authorization. So the `api_key` parameter can be omitted.
:::
